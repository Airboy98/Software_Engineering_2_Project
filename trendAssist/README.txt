Dependencies - -
Required -
MySQL Community Server https://dev.mysql.com/downloads/mysql/
Java 8.0 or higher http://www.oracle.com/technetwork/java/javase/downloads/index.html
Optional -
MySQL Workbench https://www.mysql.com/products/workbench/

Instructions -
1 - Install all required dependencies
2 - Import the database files from the database folder into MySQL workbench
3 - Run the JavaFXApp.exe found in the root of the folder on Windows
OR
3 - Run the JavaFXApp.jar found in the app folder on Linux
